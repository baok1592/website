# 轻松自助建站系统

#### 介绍
后端：Thinkphp6
移动前端：uniapp
CMS:VUE + element
PC端有2种实现方案：TP视图 ， vue + nuxt.js
本仓库没提供nuxt源码，如需请加QQ群获取


#### 功能
一套代码多端展示,前后端分离,丰富模板切换
公众号小程序管理，会员功能，付费阅读，自动采集，在线支付
